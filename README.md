# Basic Tools
## usage
```bash
- git clone git@github.com:ding2650/BasicTools.git
- npm install 
- npm run package 
- install the package (dir: /release/build/xxx.dmg|xxx.exe)
- use shortcut keys `command + Shift + F ` or click dock selection to open cliboard history
```
